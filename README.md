# 🌐 Three.js + Vite Projekt

Dies ist ein einfacher 3D-Projekt mit [Three.js](https://threejs.org/) und [Vite](https://vitejs.dev/).  
Es zeigt eine animierte 3D-Szene im Browser (z. B. eine drehende Kugel).
